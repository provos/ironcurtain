# OpenRouter provider-profile integration (docs/designs/openrouter-integration.md)

Multi-gate feature (G1-G13). Routes Docker agents (Claude Code / Codex / Goose) through OpenRouter via a named provider-profile registry in `modelProviders`. Config-shape + selection-surface change; MITM transform semantics fixed as D1-D8.
